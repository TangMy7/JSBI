# JSBI（生产看板/报表前端）

一个用于生产数据展示/统计分析的 Web 项目，前端基于 **Vue 2 + Vue CLI** 实现；后端能力由 **Python** 与 **Node.js** 提供（本仓库内包含用于本地预览/联调的后端程序）。项目中的报表页面需要配合第三方报表系统（例如 **润乾 Runqian**）或同类服务一起使用。

## 功能概览
- **大屏/看板展示**：实时数据、告警、排名等可视化页面
- **统计汇总**：多维度汇总分析页面
- **报表集成**：通过 URL/报表服务加载报表（需外部系统支持）

## 技术栈
- **前端**：Vue 2、Vue Router、Vuex、Element UI、ECharts、Axios/Fetch
- **后端（本地预览/联调）**：Python、Node.js（具体以 `js_demo/` 内程序为准）
- **报表系统**：润乾（Runqian）等（外部部署，不随仓库提供）

## 目录说明
- `src/`：前端源码（Vue2）
- `public/`：静态资源
- `js_demo/`：本地预览/联调用后端程序与脚本（包含 `522app` 等）

## 环境要求
- Node.js（建议 16+；以你的本机/CI 实际为准）
- npm
- （可选）Python 3.x（用于运行本地预览后端）

## 本地快速预览（推荐流程）
本项目的部分页面依赖后端接口数据，建议按以下顺序启动：

### 1) 启动后端（`js_demo` 内的 `522app`）
在你本地的 Python 环境中进入 `js_demo/`，启动 `522app`（用于提供接口数据/联调）。

> 注：`js_demo` 中可能还依赖数据库或第三方服务（以代码里的配置为准）。如果接口连不上，先确认后端已启动且端口/地址与前端一致。

### 2) 安装前端依赖
```bash
npm install
```

### 3) 启动前端开发服务
```bash
npm run serve
```

## 构建（生产）
```bash
npm run build
```

## 报表页面说明（润乾等）
部分页面通过 `baseUrl` 指向报表系统地址（例如润乾的 `showReport.jsp`），需要你在目标环境部署并配置好报表系统后才能正常访问。
